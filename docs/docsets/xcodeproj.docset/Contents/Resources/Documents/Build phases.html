Build phases

  • BuildPhase

    Enum that encapsulates all kind of build phases available in Xcode.

    • sources: sources.
    • frameworks: frameworks.
    • resources: resources.
    • copyFiles: files.
    • runScript: scripts.
    • headers: headers.
    • carbonResources: build legacy Carbon resources.
